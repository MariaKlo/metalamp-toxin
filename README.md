# toxin
Учебный проект от комании Metalamp. Состоит из двух частей: ui-kit и сайт toxin. При открытии ссылки на github-pages, вы увидите ссылки на все страницы.
<br>
<h3>Github Pages</h3>
<p>Ознакомиться с сайтом можете по ссылке на Github pages:</b> https://mariaklo.github.io/metalamp-toxin/</p>
<br>
<h3>NPM команды:</h3>
<b>npm i</b> - скачать все пакеты и модули.
<br>
<b>npm run dev</b> - запустить режим "development".
<br>
<b>npm run build</b> - запустить режим "production".
<br>
<b>npm run start</b> - запустить localhost:3000.
<br>
<br>npm run deploy</b> - загрузить файлы из папки dist в ветку gh-pages для запуска Github Pages.
<br>

<h3Структура проекта:</h3>
<ul>
  <li>.git - папка локального репо</li>
  <li>assets/fonts/montserrat - папка хранения шрифта</li>
  <li>dist - папка со сборкой проекта</li>
  <li>node_modules - папка со всеми загруженными пакетами и плагинами</li>
  <li>src - папка с основными файлами: assets (хранятся шрифты, изображения, глобальные стили), components (компоненты проекта, которые можно переиспользовать в разных местах проекта), layout (шаблоны страниц), pages (основные страницы ui kit и toxin)</li>
  <li>.gitignore - git игнорирует папки, указанные здесь</li>
  <li>package-lock.json - указаны все зависимости</li>
  <li>package.json - содержится основная информация о проекте, в том числе npm команды</li>
  <li>webpack.config.js - основной конфиг вебпака, development mode</li>
  <li>webpack.prod.js - production mode для вебпака</li>
</ul>

<h3>The project's structure:</h3>
<ul>
  <li>.git - local repo folder</li>
  <li>assets/fonts/montserrat - keeps font</li>
  <li>dist - the folder for compiled files</li>
  <li>node_modules - the folder with all downloaded packages and plugins</li>
  <li>src - folder with main files: assets (keeps fonts, images, global styles), components (components which you can use in other parts of the project), layout (pages layout), pages (pages of ui kit и toxin)</li>
  <li>.gitignore - git ignores folders mentioned here</li>
  <li>package-lock.json - contains all dependencies</li>
  <li>package.json -main info about project, including npm commands</li>
  <li>webpack.config.js - main webpack config, development mode</li>
  <li>webpack.prod.js - production mode for webpack</li>
</ul>
